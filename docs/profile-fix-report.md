## 页面内容详情

### 用户协议 (terms.tsx)

- **8 条主要条款**
  1. 服务说明（服务内容、服务对象）
  2. 账号注册与使用（注册资格、账号信息、账号安全）
  3. 用户行为规范（禁止内容、禁止行为）
  4. 隐私保护
  5. 知识产权（应用内容、用户内容）
  6. 免责声明（服务内容、用户行为、第三方服务）
  7. 协议变更
  8. 联系我们

### 隐私政策 (privacy.tsx)

- **8 章内容**
  1. 隐私保护承诺
  2. 我们收集的信息（主动提供、自动收集、权限调用）
  3. 信息的使用
  4. 信息的共享与披露（向用户公开、与服务提供商共享、法律披露）
  5. 信息的存储与保护（存储地点、存储期限、安全措施）
  6. 您的权利（访问更正、删除、注销、撤回同意、投诉举报）
  7. 未成年人保护
  8. 政策更新
  9. 联系我们

### 社区准则 (community.tsx)

- **7 章内容**
  1. 我们的承诺（真实可靠、尊重包容、安全私密、积极健康）
  2. 基本规则（尊重他人、真实身份、适当内容）
  3. 隐私与边界（个人信息保护、线下见面安全）
  4. 禁止行为（骚扰霸凌、虚假身份、隐私侵犯、商业营销、违法行为）
  5. 执行措施（违规处理、举报机制）
  6. 健康社交建议（建立良好印象、有效沟通、处理拒绝）
  7. 联系我们

### 黑名单 (blacklist.tsx)

- **功能特性**
  - 显示已屏蔽用户列表（头像、昵称、屏蔽时间）
  - 支持解除屏蔽操作（带确认弹窗）
  - 空状态显示
  - 如何屏蔽说明
